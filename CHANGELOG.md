# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

- Added public README documentation, CI, and release workflow hardening.
- Added deterministic asset discovery and output ordering.
- Added Python 3.10 support.
- Added builder and CLI test coverage.

## [0.2.0] - 2026-05-12

- Added `svpbuild` CLI for compiling Stardew Valley Content Patcher portrait packs.
- Added automatic portrait discovery from flexible `assets/` folder layouts.
- Added dash-schema filename parsing for variants and Content Patcher conditions.
- Added generated `manifest.json`, `content.json`, and `config.json` output.
- Added GitHub Actions release workflow for PyPI trusted publishing.
